Attach Documents

You can attach an image of your document using Capture image or File upload. Capture image will allow you to take and attach a picture using your device's camera. If you already have an image of your document, use File upload to attach that image.

Repeat this process until all files are attached; then select Continue.

Select Browse to open a window showing the files on your device. Choose the file you wish to attach, then select Upload File.
